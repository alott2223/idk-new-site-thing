# Windows 10 Web Simulator

A web-based Windows 10 interface simulator built with HTML, CSS, and JavaScript. This project provides a fully interactive desktop environment that mimics the look and feel of Windows 10.

## Features

- **Desktop Environment**: Windows 10-style desktop with gradient background
- **Start Menu**: Interactive start menu with search box and application tiles
- **Taskbar**: Bottom taskbar with start button, running apps, and system clock
- **Window Management**: Full window controls including:
  - Minimize, maximize, and close buttons
  - Drag windows to reposition
  - Multiple windows open simultaneously
  - Z-index management (click to bring to front)
- **Applications**:
  - **File Explorer**: Browse files and folders with navigation toolbar
  - **Notepad**: Text editor with toolbar
  - **Calculator**: Fully functional calculator with basic operations (+, -, ×, ÷)
  - **Browser**: Simulated web browser interface

## How to Use

1. **Open the simulator**: Open `index.html` in a web browser
2. **Launch applications**: 
   - Double-click desktop icons
   - Click the Start button and select an app from the menu
3. **Use the calculator**: Click numbers and operators to perform calculations
4. **Type in Notepad**: Click the text area and start typing
5. **Manage windows**: 
   - Click and drag title bars to move windows
   - Use minimize/maximize/close buttons
   - Click taskbar buttons to restore minimized windows

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and Windows 10 theme
- `script.js` - Interactive functionality and window management

## Technologies

- HTML5
- CSS3 (with modern features like backdrop-filter)
- Vanilla JavaScript (no frameworks required)

## Legal Note

This is a visual simulator only and does not run actual Windows 10 or any Microsoft software. It's a web-based interface that mimics the appearance and basic functionality of Windows for educational and demonstration purposes.
