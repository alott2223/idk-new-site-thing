ℹ️ About

This is a fully functional Chip-8 emulator written in JavaScript. The Chip-8 is an interpreted programming language that was used on 8-bit microcomputers in the 1970s.

Features:

  • Full Chip-8 instruction set
  • 60Hz display refresh rate
  • Sound support
  • Adjustable CPU speed
  • Load custom ROMs
